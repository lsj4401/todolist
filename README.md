# todolist
build : gradle 4.8

framework : Spring boot 2.0.4.RELEASE

unit-test : spock 1.1

DB : h2

view : angularjs 1.7, freemarker, jquery




# build
```
$ ./gradlew clean build
```

# start 
```
$ java -jar ./build/libs/todolist-0.0.1-SNAPSHOT.jar
```

# use
```
http://localhost:8080/view
```
